# Insurance Analytics Models

This repository contains a collection of Python-based analytical models designed to demonstrate practical applications of data analytics in insurance operations.

The projects focus on three important areas within insurance:

## 1. Claims Workflow Automation

This model demonstrates a simple rule-based claims triage system used to classify claims based on risk indicators such as claim amount and historical patterns.

The objective is to simulate how automation can support claims teams in prioritising high-risk cases while enabling faster processing of low-risk claims.

## 2. Underwriting Risk Segmentation

This project develops a simplified underwriting risk scoring framework.  

Policies are classified into risk categories based on factors such as:
- Industry risk score
- Previous claims history
- Claim amount

The model illustrates how risk segmentation can support underwriting decision-making and portfolio management.

## 3. Pricing Adequacy Simulation

This model demonstrates a simplified insurance pricing approach by calculating:

Expected Loss = Exposure Value × Loss Ratio

A loading factor is applied to estimate the technical premium. The model then compares the technical premium with simulated market premiums to identify policies that are:

- Underpriced
- Adequately priced
- Overpriced

This helps illustrate pricing discipline challenges in competitive insurance markets.

## Technologies Used

The models were built using:

- Python
- Pandas
- NumPy
- Matplotlib

## Purpose of the Repository

The objective of this repository is to explore how simple analytical models can support decision-making in insurance operations, including claims management, underwriting risk assessment, and pricing analysis.
